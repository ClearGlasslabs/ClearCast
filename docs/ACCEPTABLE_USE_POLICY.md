# Acceptable Use Policy

## Purpose
Define acceptable use of ClearGlassInc Artemis platforms, services, repositories, and documentation systems.

## Requirements
1. Users must access only authorized resources.
2. Users must protect credentials and access tokens.
3. Users must not perform unauthorized scanning, exploitation, or disruption.
4. Users must report suspected compromise immediately.

## Enforcement
Violations may result in immediate access revocation, investigation, and legal escalation where applicable.
