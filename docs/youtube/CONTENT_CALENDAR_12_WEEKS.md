## PRODUCTION BATCH SCHEDULE

To maintain the above schedule without burning out, use this batch recording system:

### Weekly Recording Session 1 (Monday, 3–4 hours)
- Record this week's long-form video
- Record next week's Artemis Report or series episode

### Weekly Recording Session 2 (Wednesday, 2 hours)
- Record all 3 Shorts for the current week
- Record 1 Short in advance for the following week

### Editing Pipeline (outsource when budget allows)
- Edit long-form: 2–3 hours (or 4–6 hours outsourced to an editor)
- Edit Shorts: 20–30 min each
- Edit series: 1–2 hours

### Thumbnail Creation (30 min/video)
- Template: build once in Canva or Figma, swap face/text per video
- Create thumbnail before recording to ensure correct expression and framing
- A/B test: create 2 thumbnail variants for every long-form video
